# Mi Portafolio

Este es mi portafolio personal. Lo hice con Angular para mostrar mis proyectos y que la gente pueda contactarme.

## Ver en vivo

https://portafolio-derwin.netlify.app

## Tecnologías

- Angular 17
- TypeScript
- SCSS
- EmailJS (para el formulario de contacto)

## Cómo correrlo localmente

```bash
npm install
npm start
```

## Sobre el proyecto

Es un sitio simple con:
- Información sobre mí
- Mis proyectos de GitHub (se cargan automáticamente)
- Un formulario de contacto
- Modo oscuro/claro
- Soporte en español e inglés
